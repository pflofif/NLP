
## Поля

| Поле | Метод   | Словник                   | Формат value                  |
| -------- | ------------ | -------------------------------- | ----------------------------------- |
| AMOUNT   | regex        | currencies.json                  | float + currency (UAH/USD/EUR)      |
| DATE     | regex + dict | months_ua.json, weekdays_ua.json | підтип + дата або null |
| LOCATION | dict match   | locations_lviv.json              | канонічна назва       |

---

## 1. AMOUNT

**Що вважаємо:** числова величина + явний токен валюти (`грн`, `гривень`, `₴`, `UAH`, `USD`, `EUR`).

**НЕ вважаємо:** `100%` (відсоток), `5/5` або `10 з 10` (рейтинг), число без валюти, кількість страв.

| Текст                            | Результат      |
| ------------------------------------- | ----------------------- |
| `85 грн.не вартує`       | value=85.0 UAH          |
| `200грн` (без пробілу) | value=200.0 UAH         |
| `1 711 грн`                      | value=1711.0 UAH        |
| `100% рекомендую`         | не витягуємо |
| `10 з 10`                          | не витягуємо |

---

## 2. DATE

**Підтипи:**

- `DATE_ABS` — `12.03.2025` → YYYY-MM-DD
- `DATE_MONTH` — `у березні`, `лютого 2025`
- `DATE_WEEKDAY` — `у суботу` → SAT
- `DATE_RELATIVE` — `сьогодні` → TODAY, `вчора` → YESTERDAY

**НЕ вважаємо:** тривалості (`30 хвилин`, `2 години`), час (`о 19:00`), сезони (`влітку`).

| Текст                                  | Результат                    |
| ------------------------------------------- | ------------------------------------- |
| `Були в суботу ввечері` | DATE_WEEKDAY: SAT                     |
| `Сьогодні зайшли`           | DATE_RELATIVE: TODAY                  |
| `відвідали 12.03.2025`           | DATE_ABS: 2025-03-12                  |
| `чекали 30 хвилин`            | не витягуємо               |
| `у суботу.Чесно`              | span не включає крапку |

---

## 3. LOCATION

**Що вважаємо:** топоніми Львова зі словника `locations_lviv.json` (70 записів): площі, вулиці, райони, пам'ятки, варіанти назви міста. Пошук case-insensitive, greedy.

**НЕ вважаємо:** `"центр"` без Львова, назви інших міст, загальні вирази (`"десь на вулиці"`).

| Текст                                   | Результат                       |
| -------------------------------------------- | ---------------------------------------- |
| `у Львові це мій топчик` | Львів                               |
| `дахи старого міста`       | Старе місто                    |
| `на ПЛОЩІ РИНОК`               | площа Ринок (case-insensitive) |
| `це самий центр`               | не витягуємо                  |
| `ресторан Підзамче`        | Підзамче (context-agnostic)      |
