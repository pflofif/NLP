# Lab 2

## 1. Задача та дані

- **Напрямок:** C (Retrieval / Пошук закладу за відгуком).
- **Text Field:** Поле `text` містить відгуки користувачів про заклади Львова (Google Maps reviews).
- **Зміст:** Суміш української мови, локального сленгу ("зупа", "штрудель"), емодзі, номерів телефонів та адрес.

   - Змонтує дані (або завантажить `processed.csv`).

## 3. Правила обробки (Policy)

Пайплайн реалізовано у модулі `src/preprocess.py`. Основні кроки:

- **Masking PII:** Заміна URL-адрес, email та номерів телефонів на теги `<URL>`, `<EMAIL>`, `<PHONE>`.
- **Normalization:** Приведення до нижнього регістру та уніфікація всіх видів апострофів до стандарту `’` (U+2019).
- **Cleaning:** Видалення зайвих пробілів, табуляцій та символів пунктуації, що не несуть змістовного навантаження.
- **Segmentation:** Розбиття тексту на речення за допомогою `tokenize_uk`.

## 4. Топ-5 "болючих" Edge Cases

1. **Варіативність апострофа:** Написання "Львів'янин", "Львів`янин" та "Львів'янин" — тепер це один токен.
2. **Склеєна пунктуація:** "Смачно.Дуже" або "Привіт,Львів" — коректно розділяються на окремі слова.
3. **Емодзі:** "Рекомендую 😍😍😍" — емодзі видаляються, не залишаючи зайвих пробілів.
4. **Маскування тех-даних:** Посилання на Google Maps у тексті замінюються на url, що запобігає засміченню словника.
5. **Сміття в кінці:** Видалення залишків системних символів `\r\n`, які часто трапляються у вивантаженнях.
